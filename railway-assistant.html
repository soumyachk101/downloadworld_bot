🔐 Security Best Practices

  • Never commit .env file to GitHub
  • Use strong random BOT_TOKEN
  • Rotate COOKIES_B64 every 2 weeks
  • Set ADMIN_ID to restrict admin commands
  • Enable 2FA on Telegram and GitHub

🐛 Common Issues

Instagram rate limit / login required

Set COOKIES_B64 environment variable with your cookies.

See README-COOKIES.md for step-by-step instructions.

Bot not responding

Check: BOT_TOKEN is set correctly (format: 123456:ABC...)

Check logs: railway logs

Health check failing

Ensure PORT is set to 3000 (Railway sets this automatically).

Check logs for port binding errors.
